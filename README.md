# NASA Earth Viewer

![NASA Earth Viewer](H:\SoC%20Projects\Portfolio\images\NasaExplorer-Thumb.png)

A React application that lets you explore Earth imagery captured from space. Built during a hackathon as part of the School of Code Week 8 React Deep Dive.

## Features

- Select any date to fetch a real photograph of Earth taken by NASA's DSCOVR satellite via the EPIC API
- View J2000 positional coordinates for the DSCOVR spacecraft, the Sun, the Moon, the galactic centre, and Earth
- Toggle coordinate panels on/off using checkboxes
- Animated space background with NASA branding

## Tech Stack

- React 18
- NASA EPIC API

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## API

This project uses the [NASA EPIC API](https://epic.gsfc.nasa.gov/about/api) to retrieve Earth Polychromatic Imaging Camera imagery and positional metadata by date.
